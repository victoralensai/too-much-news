# TOO MUCH NEWS!!

An overwhelming, real-time news "firehose" designed to visualize the sheer volume of global information.

Created by **Victor Jean** (vctor.me) as an artistic/humorous exploration of information overload.

## The Concept
Inspired by the feeling of adding too many RSS feeds to an aggregator and realizing you can never keep up. Instead of a useful tool, this is a visualization of the chaos of global news production.

## Tech Stack
- **Backend:** Node.js, TypeScript, Socket.io
- **Polling:** RSS-Parser (cycling through ~65 global sources)
- **Frontend:** Vanilla JS, HTML/CSS (Matrix-inspired Terminal theme)

## Quick Start
1. Install dependencies: `npm install`
2. Build: `npm run build`
3. Start: `npm start`
4. Visit: `http://localhost:3000`
